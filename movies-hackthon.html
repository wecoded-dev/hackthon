Movies Website Hackthon


AquaScript Movies Website Hackthon



Description

Build a beautiful, creative, and fully responsive movies webpage by using AquaScript Movies Data API. Your mission is to showcase your frontend skills by designing a stunning interface that fetches and display movies data in the way that look modern, clean and professional.


Rules

Your submission website will be responsive and the code will be clean. Avoid to use the AI because this challenge is engaged to level up your coding skills. A user can submit more than one submissions. You cannot use the Movies Data API except AquaScript API. You will make the website on Github because you will also submit repository link. We will not consider submission after due date.

Prizes

We will announce three winners. First winner will get the bundle of 300 Freepik (PRO) & Other Vector Images, Stickers, Icons, etc. and 10 (PRO) websites templates & one advertisement on AquaScript for free & one custom domain for a year. Second winner will get the bundle of 200 Freepik (PRO) & Other Vector Images, Stickers, Icons, etc. and 7 (PRO) websites templates & one advertisement on AquaScript for free. Third winner will get the bundle of 100 Freepik (PRO) & Other Vector Images, Stickers, Icons, etc. and 4 (PRO) websites templates & one advertisement on AquaScript for free.
